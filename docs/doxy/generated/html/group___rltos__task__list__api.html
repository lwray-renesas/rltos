◆ list_index_t

enum list_index_t

Provides enumerated type to index task item lists, This is used to index between a tasks state list (running or idle) and an auxilliary list for an object which the task is associated with waiting on.

Enumerator
state_list 

Index of list where the item exists in the running or idle list

aux_list 

Index of list where the item exists waiting on an object
